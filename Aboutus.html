This web application was developed as part of a web development assignment to demonstrate the integration of front-end and back-end technologies. The system allows users to register, log in, and securely access their profile and a contact form. It includes client-side validation using JavaScript and handles data processing and session management with PHP. The design is fully responsive and user-friendly, built using semantic HTML5 and styled with modern CSS techniques like Flexbox and Grid. This project showcases core concepts in web development including form handling, user authentication, data storage, and protected routes.
